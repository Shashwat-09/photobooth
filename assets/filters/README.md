# Filter Assets

This folder contains overlay images used for photo filter effects.

## Required Assets

Replace the placeholder files with final PNG assets:

### 1. filmgrain.png
- **Purpose**: Tileable film grain texture
- **Recommended Size**: 1024×1024px
- **Format**: PNG with alpha channel
- **Usage**: Tiled across the image for grain effect
- **Notes**: Should be seamless/tileable for best results

### 2. lightleak.png
- **Purpose**: Light leak overlay effect
- **Recommended Size**: 1024×1024px
- **Format**: Transparent PNG
- **Usage**: Overlayed with screen blend mode
- **Notes**: Warm orange/yellow tones work best

### 3. vignette.png
- **Purpose**: Radial vignette (dark edges)
- **Recommended Size**: 1024×1024px
- **Format**: PNG with alpha channel
- **Usage**: Radial gradient from transparent center to black edges
- **Notes**: Should fade smoothly from center to edges

### 4. dust.png
- **Purpose**: Scratches and dust particles overlay
- **Recommended Size**: 1024×1024px
- **Format**: Transparent PNG
- **Usage**: Overlayed with screen blend mode
- **Notes**: White scratches/dust on transparent background

### 5. polaroid-frame.png (Optional)
- **Purpose**: Polaroid border frame
- **Recommended Size**: 1024×1024px
- **Format**: PNG with transparent center
- **Usage**: Frame overlay for polaroid effect
- **Notes**: White border with transparent center area

## Creating Assets

You can create these assets using:
- Photoshop/GIMP for manual creation
- Online texture generators for grain
- Free texture sites (e.g., textures.com, freepik.com)
- AI image generators for light leaks and vignettes
